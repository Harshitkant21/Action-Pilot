# System Architecture

## Overview

ActionPilot follows a modular client-server architecture designed to support AI-driven planning, monitoring, risk analysis, and execution assistance.

The architecture prioritizes:

* Rapid development
* Scalability
* Clear separation of concerns
* Agent-driven workflows
* Background task processing

---

# High-Level Architecture

```text
React Frontend

↓

Express Backend

↓

PostgreSQL Database

↓

Gemini API

↓

Redis + BullMQ

↓

Notification Service
```

---

# Scheduled Jobs

## Daily Standup Job

Runs every morning.

Responsibilities:

* Generate daily summaries
* Prepare execution recommendations
* Create standup messages

---

## Risk Evaluation Job

Runs periodically.

Responsibilities:

* Evaluate active goals
* Calculate risk scores
* Identify delayed work

---

## Inactivity Monitoring Job

Runs periodically.

Responsibilities:

* Detect inactive users
* Detect stalled goals
* Trigger follow-ups

---

## Recovery Planning Job

Runs when risk exceeds threshold.

Responsibilities:

* Generate revised plans
* Suggest scope adjustments
* Reprioritize work

# Agent Communication Model

Agents do not communicate directly with each other.

The backend acts as the orchestration layer.

```text
Frontend

↓

Backend

↓

Agent

↓

Database

↓

Agent

↓

Frontend
```

This approach keeps the system simple and maintainable during the hackathon.

## Why React

Fast development and strong ecosystem.

## Why Express

Simple, lightweight, and suitable for rapid iteration.

## Why PostgreSQL

Reliable relational database with strong support for structured data.

---

# Architecture Summary

ActionPilot uses a modular AI-driven architecture where specialized agents collaborate through a centralized backend to help users plan, execute, monitor, and complete goals before deadlines are missed.

The system combines real-time interactions, background monitoring, AI reasoning, and proactive notifications to create a true execution companion rather than a traditional task management application.
